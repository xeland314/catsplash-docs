`catsctl` es la interfaz de línea de comandos para administrar Catsplash.

## Uso general

```bash
sudo catsctl [opciones] <comando> [argumentos]
```

## Opciones

| Opción | Descripción |
|---|---|
| `--config <path>` | Ruta al archivo de configuración TOML |

## Comandos

### `status`

Muestra estadísticas del sistema: interfaces, puerto, sesiones activas/pendientes y tráfico total.

```bash
sudo catsctl status
```

### `list`

Lista todos los clientes en la base de datos con su estado, MAC, IP, tiempo de conexión, datos usados, límites y ancho de banda.

```bash
sudo catsctl list
```

### `auth`

Autoriza manualmente a un cliente.

```bash
sudo catsctl auth <mac> <ip>
```

**Ejemplo:**
```bash
sudo catsctl auth aa:bb:cc:dd:ee:ff 192.168.10.50
```

### `kick`

Desconecta manualmente a un cliente.

```bash
sudo catsctl kick <mac>
```

**Ejemplo:**
```bash
sudo catsctl kick aa:bb:cc:dd:ee:ff
```

### `extend`

Extiende la sesión de un cliente por N minutos.

```bash
sudo catsctl extend <mac> <minutos>
```

**Ejemplo:**
```bash
sudo catsctl extend aa:bb:cc:dd:ee:ff 30
```

### `limit`

Establece un límite de datos para un cliente en MB. Usa `0` para eliminar el límite.

```bash
sudo catsctl limit <mac> <mb>
```

**Ejemplo:**
```bash
sudo catsctl limit aa:bb:cc:dd:ee:ff 500
```

### `band`

Establece límites de ancho de banda para un cliente.

```bash
sudo catsctl band <mac> <descarga> <subida>
```

Los valores pueden ser en `mbit`, `kbit`, etc. Usa `0` para eliminar el límite.

**Ejemplo:**
```bash
sudo catsctl band aa:bb:cc:dd:ee:ff 5mbit 2mbit
```
